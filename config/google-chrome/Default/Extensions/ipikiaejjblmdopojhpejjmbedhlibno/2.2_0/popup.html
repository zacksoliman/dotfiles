There are five ways you can use Spreed:

  1. Access the Spreed menu by clicking on the Spreed icon to the right of the address bar
  2. Select "Spreed current website"
  3. OR "Paste text into Spreed" to speed read text copy-pasted in from non-Chrome sources

OR
  1. Alt+V to automatically extract main content of webpage
OR
  1. Alt+Shift+V to start text selector
  2. Click on body of text that you want to read
OR
  1. Select text on webpage
  2. Alt+V to open Spreed with selected text
OR
  1. Select text on webpage
  2. Right click
  3. Select "Spreed selected text"
