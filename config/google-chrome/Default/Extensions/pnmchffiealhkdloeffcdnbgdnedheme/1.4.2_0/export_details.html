History Trends Unlimited: Export Details
File Format

The Raw Data Export is a tab-delimited file with the fields listed below.

Name Description Example
URL The visited URL http://scores.sports.example.com/team?id=123456
Host* The hostname of the URL scores.sports.example.com
Root Domain* The root domain of the URL according to publicsuffix.org example.com
Visit Time (ms) The visit time, in milliseconds since 01 January, 1970 1384634958041.754
Visit Time (string) The visit time as a string, in local time, with the following format:
YYYY-MM-DD HH24:MI:SS.FFF		 2013-11-16 14:49:18.041
Day of Week The day of the week for the visit time. Values are 0-6. Sunday=0, Monday=1, etc. 6
Transition Type How the browser navigated to the URL (as defined here) link
Page Title* The title of the visited URL Life, the Universe, and Everything

* These columns can be blank.
