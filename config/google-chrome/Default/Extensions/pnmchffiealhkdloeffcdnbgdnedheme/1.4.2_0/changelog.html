History Trends Unlimited: Changelog

1.4.2:  2016-01-23

  • Bug fix: Fixed HTML encoding issue on the Search page.

1.4 / 1.4.1:  2015-11-01

  • Added Auto Backup feature on the Options page.
  • Added FAQ link in the Feedback section on the Options page.
  • Bug fixes for importing and exporting.
  • The statistics displayed after importing a file now include the number of duplicate visits that were not imported (if there were any).
  • Changed text on the Export Raw Data dialog to clarify its purpose.
  • Automatically close Export Raw Data dialog when export is done.

1.3:  2015-09-07

  • Added the ability to delete visits on the Search page.
  • Bug fix: Searches that match part of the title and part of the URL now return results.
  • Bug fix: Properly encode HTML entities when displaying search results.

1.2.1:  2015-06-19

  • Bug fix for importing "Raw Data" file.

1.2:  2015-06-17

  • Begin using Chrome's built-in internationalization support to display dates, times, and numbers in the user's locale. This was previously handled by an external library that had gotten out of date.
  • Options: Added option to switch between 12 and 24-hour time.
  • Search: Fixed keyword highlighting bug.
  • Search Tips: Changed wording to indicate only ending wildcards are supported.
  • Search Tips: Removed incorrect quotes around the boolean example.

1.1.3:  2015-06-07

  • Importing now supports the "Raw Data" file format.

1.1.2:  2015-02-28

  • Bug fix: Search results now display for users with a non-US locale.

1.1 / 1.1.1:  2014-11-22

  • Added the Search page, which supports browsing your history, searching the page title and url, and viewing a specific date.
  • Options: Added ability to choose the start page (either Trends or Search).
  • Created navigation sidebar

1.0.9:  2014-11-03

  • Begin tracking page title

1.0.8:  2014-10-19

  • Removed the "Export Chrome's Archived History" instructions from the Options page. Chrome no longer maintains an Archived History database. See https://code.google.com/p/chromium/issues/detail?id=359377 for more details.
  • Fixed font issue
  • Updated the Import instructions on the Options page

1.0.6 / 1.0.7:  2013-12-01

  • Added the "Limit results to:" menu
  • Bug fix: Clicking a point on the "Visits per Day" chart now filters correctly (1.0.7)

1.0.4 / 1.0.5:  2013-11-25

  • Added Unique Visits tab
  • The "View More..." link previously displayed results 11-100. It's been renamed to "View Top 100" and displays results 1-100.
  • When viewing the Top 100 results, the columns are now sortable.
  • Bug fix: When filtering by domain, some subdomains were appearing one level higher than expected (e.g., a domain filter for "example.com" would display "foo.bar.example.com" instead of "bar.example.com").
  • Performance improvements (1.0.5)

1.0.3:  2013-11-17

  • Added "Export Raw Data" feature with 2 file format options
  • Added this Changelog page
  • Rearranged Export/Import instructions on Options page for clarity
  • Updated list of public suffixes from http://publicsuffix.org/

1.0.2:  2013-04-09

  • Add Feedback section on Options page

1.0.1:  2013-03-18

  • Minor updates to export instructions on Options page

1.0:  2013-03-17

  • Initial Release
